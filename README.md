# QuestLog App

An RPG-themed daily to-do list with a leveling system built in JavaScript.

## Overview

Turn your daily tasks into quests. Complete them to earn XP, level up your character, and track your productivity in a fun, game-like way.

## Features

- Create tasks as RPG quests with XP values
- Earn XP and level up your character by completing tasks
- Track your progress with a visual XP bar
- Quest categories: Daily, Weekly, Side Quest
- Persistent storage so your progress is saved

## Tech Stack

- **Language:** JavaScript (Vanilla)
- **Storage:** localStorage
- **Styling:** CSS with RPG-inspired theme

## Getting Started

1. Clone the repo
2. Open `index.html` in your browser
3. Start adding quests and leveling up!
